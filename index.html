Here are some of the things we learned in our 10/8/15 class.

  • Make sure to load a universal Library and your own library as "script src". eg. script src="http://ajax.googleapis.com/ajax/libs/jquery/1.10.2/jquery.min.js"
  • The Document Object Model (DOM) is a standard for interacting with and representing objects in HTML, XHTML, and XML. jQuery makes it incredibly easy to select elements in the DOM, which include practically any HTML tag or element
  • Easily selects elements
  • By calling $ in the JavaScript console, we're just using a shorthand way of accessing the jQuery library's main function, that all others are descended from.
  • The $(document).ready() snippet ensures that your code doesn't run until your document fires a ready event. The ready event fires when all of the page's DOM elements are loaded, even if multimedia elements aren't fully loaded
  • Anonymous functions are another way to write functions that makes them a "firstclass citizen" data type. They can be given as an argument to a function (yes, a function taking a function), returned by a function or block of code or assigned to a variable. This is how one is written: function(){ alert("I am anonymous!");}
  • Anonymous functions can be stored in variables var my_anon_func = function(){ alert("Har!"); }
  • Callbacks: A callback is an anonymous function supplied to another function to be run when that function is done running. Anonymous functions are often used in callbacks
  • Here are some call back examples:
    EXAMPLE 1: var someOtherFunction = function(){ alert('hello world'); }. This defines an anonymous function called "someOtherFunction". EXAMPLE 2: function someRunner(anyFunction){ console.log(2+2); anyFunction(); } This declares a regular or anonymous function that takes another function as an argument, does something,then runs that other function EXAMPLE 3: someRunner(someOtherFunction);This calls the regular function, supplying the initial anonymous function as an argument
